❓ Qu'est-ce qu'une base de données ?

Une Base de Données est un ensemble de données relatif à un domaine précis, caractérisé par être :
  • Structuré : Organisé sous forme de tables.
  • Exhaustif : Contient toutes les informations nécessaires.
  • Non redondant : Sans répétitions inutiles.
  • Persistant : Stocké de manière durable.

💻 contrainte d'integrité

L'intégrité de table (ou d'entité) : 🔑

Lintégrité des domaines :

  • contraintes des types : Chaque champ doit recevoir une donnée correspondant au type défini (ex: INT, DATE, VARCHAR)
  • Contraintes de valeurs : On utilise la contrainte CHECK pour limiter les valeurs possibles. Par exemple, dans votre TP, la note doit obligatoirement être comprise entre 0 et 20
  • Caractère obligatoire : La contrainte NOT NULL impose que certains champs, comme le nom ou le prénom, ne soient jamais laissés vides

💻Lintégrité referentielle :

Elle assure la validité des liens et de la communication entre les différentes tables de la base de données
  • Cohérence des liens : Par exemple, dans la table DEVOIR, on ne peut pas insérer un Id_elv s'il n'existe pas déjà dans la table ELEVE . Cela garantit que les relations entre les entités (Élèves, Matières, Devoirs ou Clients et Chambres) restent toujours logiques et sécurisées
