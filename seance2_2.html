📁 Exercice 1 : Création de la table ELEVE

Écrire le code SQL pour créer la table ELEVE avec les champs Id_elv (Clé Primaire)

CREATE TABLE ELEVE ( Id_elv INT PRIMARY KEY, nom VARCHAR(50) NOT NULL, prenom VARCHAR(50) NOT NULL, date_naiss DATE );

📁 Exercice 2 : Création de la table MATIERE

Écrire le code SQL pour créer la table MATIERE avec les champs Id_mat (Clé Primaire) et Nom_mat.

CREATE TABLE MATIERE ( Id_mat VARCHAR(10) PRIMARY KEY, Nom_mat VARCHAR(50) NOT NULL );

📝 Exercice 3 : Création de la table DEVOIR

Créer la table DEVOIR. Attention : Id_elv et Id_mat sont des clés étrangères, et la note doit être comprise entre 0 et 20.

CREATE TABLE DEVOIR ( Id_elv INT, Id_mat VARCHAR(10), note DECIMAL(4,2), PRIMARY KEY (Id_elv, Id_mat), FOREIGN KEY (Id_elv) REFERENCES ELEVE(Id_elv), FOREIGN KEY (Id_mat) REFERENCES MATIERE(Id_mat), CONSTRAINT chk_note CHECK (note BETWEEN 0 AND 20) );

➕ Exercice 5 : Modification (Ajout de colonne)

Ajouter l'attribut coefficient à la table MATIERE.

ALTER TABLE MATIERE ADD coefficient INT;

🔄 Exercice 6 : Modification de type

Modifier le type de l'attribut coefficient en DECIMAL.

ALTER TABLE MATIERE MODIFY coefficient DECIMAL(3,2);

❌ Exercice 7 : Suppression de colonne

Supprimer l'attribut date_naiss de la table ELEVE.

ALTER TABLE ELEVE DROP COLUMN date_naiss;
