💻 Séance 3 : Manipulation SQL (LMD)

Maîtrisez la Consultation, la Mise à jour et la Suppression des données 📊

🔍 1. Consultation des données (SELECT)

Le langage LMD permet d'interroger la base pour afficher des informations précises .

Q1 : Afficher toute la table des élèves.

SELECT * FROM ELEVE ;

Q2 : Afficher les Noms et les prénoms des élèves.

SELECT nom, prenom FROM ELEVE ;

Q3 : Afficher les noms des élèves qui commencent par « M ».

SELECT nom FROM ELEVE WHERE nom LIKE 'M%' ;

Q4 : Afficher les coordonnées de l’élève ayant l'identifiant « E002 ».

SELECT * FROM ELEVE WHERE Id_elv = 'E002' ;

Q5 : Trier par ordre décroissant les élèves selon leurs moyennes.

SELECT * FROM ELEVE ORDER BY moyenne DESC ;

🔄 2. Mise à jour des données (UPDATE)

On utilise UPDATE pour modifier des enregistrements existants .

Exercice : Ajouter plus un (+1) à toutes les notes des élèves.

UPDATE DEVOIR SET note = note + 1 ;

❌ 3. Suppression des données (DELETE)

On utilise DELETE pour retirer des lignes (uplets) spécifiques .

Exercice : Supprimer les élèves dont l'identifiant se termine par le chiffre 3.

DELETE FROM ELEVE WHERE Id_elv LIKE '%3' ;
